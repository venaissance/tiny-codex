# File Explorer — IDE-level File Management

## 目标
左侧文件区对齐 IDE 80%+ 功能。

## 功能清单
- [ ] 树形目录展示（递归展开/折叠子目录）
- [ ] 创建文件（右键菜单 / 工具栏按钮）
- [ ] 创建文件夹
- [ ] 文件重命名（双击 / F2）
- [ ] 文件删除（右键 / Cmd+D，带确认对话框）
- [ ] 文件移动（拖拽）
- [ ] 右键上下文菜单（New File, New Folder, Rename, Delete, Copy Path）
- [ ] 文件图标（按扩展名区分）
- [ ] 搜索过滤
